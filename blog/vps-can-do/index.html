2019.10.09 Upd：这篇文章弃坑啦，以后直接放在 Lab 页面里了。

这篇文章作为我的颓废成果，暨萌新颓废指南……记录下我的 VPS 上建的东西。

生命不息，折腾不止！

BWGONG 服务器配置信息：

  • Physical Location: California, US
  • Operating system: Centos 7.3 x86_64 bbr
  • RAM: 1024 MB
  • SWAP: 1283.99 MB
  • Disk (/): 20 GB

ALIYUN 服务器配置信息：

  • Physical Location: Hangzhou, CN
  • Operating system: Centos 7.3 x86_64
  • RAM: 2048 MB
  • SWAP: 1283.99 MB
  • Disk (/): 40 GB

开始驾驶！

  • 搭建 LNMP 环境，建设网站；
    LNMP：Linux, Nginx, MySQL, PHP（强烈推荐 Appnode 或者 LNMP.org）

    • 部署 WordPress 等网站博客平台（就是这里）；
    • 搭建私有云盘（KODExpolorer / OwnCloud / NextCloud）；
    • 搭建一个论坛（Flarum BBS）；
    • 托管一个图床（虽然我觉得有 WordPress 了没必要）；
    • 托管各种网站程序，例如 Core Value Encoder 等等（？？？）；
    • 甚至可以托管一些网页游戏（比如 A dark room）；
    • 文件分享系统（定时删除、阅后即焚）；
    • 文字阅后即焚（信息安全防查表的最佳解决方案）；
    • 在线 IDE（没准用得上）；
    • 音乐播放器（Adams 主题作者 Tokinx 写的，颜值超高）；
    • 网页采集器（不会 python 爬虫，于是……）；
    • nginx 反向代理建设 Google 镜像站（以备不时之需……）；
    • nodeppt（比什么巨硬的 PowerPoint 不知道高到哪里去了）；
  • 小飞机服务端；
  • 作为 FTP 服务器存储文件（其实有 KOD Cloud 了这个没必要）；
  • Minecraft 服务器；
  • hack.chat 等即时匿名聊天系统；
  • 域名邮箱（me@skywt.cn）；
  • 作为 Git 服务器托管代码（甚至可以搭建一个 GitLab）；
  • 利用 Aria2+AriaNg/WebUI-Aria2+Rclone 搭建网盘离线下载；
  • 用 Docker 运行 CoolQ，搭建 QQ 机器人；
  • 最重要的用途：装逼；
  • 未完待续…
