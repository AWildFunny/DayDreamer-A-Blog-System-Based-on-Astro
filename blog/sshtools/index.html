各个平台（Windows/Linux/Android）SSH 连接工具

2018 年 1 月 14 日 08:40

文章目录

  • Linux
  • Windows
  • Android

最近买了个VPS，CentOS，不提供图形界面……那么正确的打开方式当然就是SSH连接了~

Linux

Linux 下的 SSH 连接最方便了，大多数 Linux 发行版都自带 openSSH 。我用的是 Ubuntu16.04，终端里直接有SSH的命令：

ssh root@127.0.0.1 -p 8080

其中 root 就是那个 VPS 登录的用户名，127.0.0.1 就是 IP 地址，8080 是端口。如果不使用 -p 参数，默认端口就是 22。如果有密码，接下来就会要求输入密码。然后就算是连上了。

如果没有安装SSH服务，可以通过以下命令安装：

sudo apt-get install openssh-client
sudo apt-get install openssh-server

Windows

Windows 系统里没有自带的 SSH，下载安装 openSSH，就可以在 cmd 里和 Linux 一样运行SSH命令了……

Android

Android 有很多第三方 SSH 软件，比如我比较喜欢的 JuiceSSH。还有各种终端模拟器都是自带 SSH 的。
