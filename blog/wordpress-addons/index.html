14个自用超级强大的 WordPress 插件推荐

2018 年 08 月 16 日
共 6 条评论

⚙️ 全新博客前端仍在测试中。如果文章显示不正常，可以尝试访问原博客的这篇文章。

一入 WordPress 深似海……众所周知 WordPress 的一大强大之处就是拥有一大堆神奇的插件。今天介绍下我私藏的几个超级强大的插件 ~

Markdown 编辑：WP Editor.md

强烈推荐！！！
如作者所说，“或许这是一个 WordPress 中最好，最完美的 Markdown 编辑器”。国人开发的，和洛谷上的很像，是开源的。支持数学公式（而且是解析超快的 KaTeX），支持 Mermaid 流程图（下次遇到图论题目的题解都不用传图片了，直接画 Mermaid 流程图），甚至支持 Emoji！

链接：WP Editor.md

2019XXXX 更新：这个插件现在和 WordPress 5.0 编辑器不兼容，要用 classic editor，并且开发者已经停止开发……不过仍然是目前 WordPress 最完美的 markdown 编辑器。

文章索引：Table of Contents Plus

和上面一个配套使用，可以显示文章目录。本文前面的文章目录就是用这个生成的。自定义的设置也很强大。

Table of Contents Plus

（本博客主题似乎与这个插件不兼容，出了点微小的 bug……）

搜索引擎优化：多合一SEO包（All in One SEO Pack）

做搜索引擎优化用的，有生成 XML 站点地图（似乎 Google、百度都可以用）、生成摘要、自定义页面关键词等等一大堆功能。对 Google 十分友好。

All in One SEO Pack

生成缓存：WP Super Cache

这个是网站速度优化用的，可以自动生成网站缓存，网站访问速度会明显提升。

WP Super Cache

插入代码：Header and Footer Scripts

很实用的一个插件，可以在所有页面的头部（header）和页脚（footer）插入代码。用这个插入百度自动推送代码以及谷歌 AdSense 自动广告代码就很方便，不需要去改主题了。

Header and Footer Scripts

插入 PHP 代码：PHP Code Widget

又是很方便的一个插件，这个可以和下面的 Elementor 和 WP-PostViews 配合使用，提供了 php 代码的小工具。

PHP Code Widget

禁止谷歌字体：Disable Google Fonts

WordPress 默认是使用 Google 的 Open Sans 互联网字体的，但是因为神奇的原因，中国大陆会出现问题（如果使用默认主题的 WordPress 什么都不装，会发现访问什么页面都要加载很长时间，这就是一直等待谷歌服务器响应而等不到导致的）。这个插件就可以禁用谷歌字体，改用系统默认字体（部分主题还提供了自定义字体的选项）。

Disable Google Fonts

页面编辑：Elementor

超级喜欢的一个插件！可以用可视化的编辑器自定义某个页面，自定义的选项超级丰富。个人感觉 WordPress 最近推出（并且“强制”使用）的古腾堡，就是个 Elementor 的阉割版……用 Elementor 可以创建各种东西，比如说什么社交图标，什么功能菜单……简直就是想怎么做就怎么做，比古腾堡不知道高到哪里去了。本网站的首页就是用 Elementor 做的。（现在已经不是了……）
有收费的 Pro 版，但是免费版已经够良心了。

Elementor Page Builder

20181011：已经弃用该插件，开始养老……（不过这的确是个很好的插件）

各种功能：Orbit Fox Companion

这个小插件提供了很多功能，官方说：“This swiss-knife plugin comes with a quality template library, menu/sharing icons modules, and newly added Elementor/BeaverBuilder page builder widgets on each release.”
我主要用到的就是分享按钮，似乎比 AddThis 要快一点（这些按钮纯粹好看，我知道没人真的用那几个按钮……）和 Elementor 的额外插件，还有菜单图标功能。这个菜单图标很厉害，集成了一大堆图标。我的网站上面一行菜单的图标就是用这个搞的。还有什么网站分析之类的功能，总之也不错了。

Orbit Fox Companion

20181011：已经弃用该插件，开始养老……（不过这也的确是个很好的插件）

文章访问统计：WP-PostViews

这个插件可以统计每篇文章的访问量。结合上面 PHP Code Widget 的插件，用 the_view() 函数可以取得当前文章访问量。有些主题是默认显示的。

WP-PostViews

分享按钮：Share Buttons by AddThis

这个社交分享按钮很强大，主要是作为一个国外的软件，居然囊括了几乎国内的所有社交平台！还附带什么分析之类的功能。不过有时候加载速度挺慢的（毕竟人家国外的），我该用 Orbit Fox Companion 的分享按钮了。
顺便提一句，国内有个类似的社交分享按钮插件，AddThis 的模仿者（十分丑陋，还停留在上个年代的拟物风），叫做 JiaThis……

Share Buttons by AddThis

文章批量修改替换：Search & Replace

这个插件可以在所有文章里查找指定关键词并统一替换，还提供了备份数据库的功能。

Search & Replace

登陆页面编辑：DX Login Register

这个东西似乎是国人开发的，可以自定义 WordPress 登陆页面的一些东西，比如说图标、提示语之类的。每次看见一个硕大的 WordPress 图标就想要换掉……

DX Login Register

管理后台编辑：Admin Menu Editor

装了上面这么多插件，你会发现管理后台的菜单杂乱无章：有些插件设置是“设置”菜单里的，有些是“工具”菜单里的，有些把自己放到“插件”菜单里，有些甚至独立成一个菜单……这是坠痛苦的……
Admin Menu Editor 这个插件就可以编辑后台管理界面的菜单了。我新增一个“插件设置”菜单，WordPress 插件的设置一律放里面。看起来就很舒服了 ~

Admin Menu Editor

一入 WordPress 深似海，从此学习是路人我变得更加热爱学习！！！
