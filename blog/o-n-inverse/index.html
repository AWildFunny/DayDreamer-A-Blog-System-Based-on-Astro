以 O(N) 线性时间复杂度递推逆元的方法

2018 年 8 月 9 日 06:54

文章目录

  • 原理与证明
  • 代码

在之前我们已经知道乘法逆元的三种求法，对于一般的题目让你把答案模一个质数，如果要求逆元一般用费马小定理，可以在 Θ(Nlog2(N))\Theta (Nlog_2(N))Θ(Nlog2​(N)) 时间内构造出 1 到 N 的逆元：inv(x)=xmo−2 mod moinv(x)=x^{mo-2} \bmod moinv(x)=xmo−2modmo。但是对于 10710^7107 级别的 NNN，这样的求法就显得有点慢。能不能在 Θ(N)\Theta (N)Θ(N) 时间内递推出 inv(x)inv(x)inv(x) 呢？

答案是肯定的。

线性推逆元的递推式

（模数是 NNN，⌊x⌋\lfloor x \rfloor⌊x⌋ 表示 xxx 向下取整）

inv(i)=(N−N/i)∗inv(N mod i) mod Ninv(i)=(N-N/i)\ast inv(N\bmod i)\bmod Ninv(i)=(N−N/i)∗inv(Nmodi)modN

原理与证明

在某位大佬的博客上看到一个证明。（以下 a/ba/ba/b 与 ⌊ab⌋\displaystyle \lfloor \frac a b \rfloor⌊ba​⌋ 均表示 aaa 整除 bbb ）

假设 k=⌊Ni⌋,b=N mod i\displaystyle k=\lfloor \frac N i \rfloor ,b=N \bmod ik=⌊iN​⌋,b=Nmodi；显然有：

k∗i+b≡0(modN)k \ast i + b \equiv 0 \pmod Nk∗i+b≡0(modN)

变换得到：

−k∗i≡b(modN)-k \ast i \equiv b \pmod N−k∗i≡b(modN)

两边同除 i∗ki \ast ki∗k：

−k∗inv(b)≡inv(i)(modN)-k \ast inv(b) \equiv inv(i) \pmod N−k∗inv(b)≡inv(i)(modN)

把 kkk 和 bbb 换回来，就得到了！

$$

  • \lfloor \frac N i \rfloor \ast inv(N \bmod i)\equiv inv(i) \pmod N \
    inv(i) \bmod N = - (N/i) \ast inv(N \bmod i) \bmod N \
    $$

因为 inv(i)inv(i)inv(i) 有很多个，我们可以求出最小的一个，一定小于等于 NNN（证明详见：乘法逆元的三种求法），所以左边的  mod N\bmod NmodN 可以去掉了。右边有个  mod N\bmod NmodN ，为了让右边大于 0，我们可以给它加上 NNN。

inv(i)=(N−N/i)∗inv(N mod i) mod Ninv(i) = (N-N/i) \ast inv(N \bmod i) \bmod Ninv(i)=(N−N/i)∗inv(Nmodi)modN

初始 inv(0)=1inv(0)=1inv(0)=1，可以愉快地递推啦～

代码

其实知道了递推式，代码就两行……

    inv[0]=1;
    for (int i=1;i<=M;i++) inv[i]=(N-(N/i))*inv[N%i]%N;
